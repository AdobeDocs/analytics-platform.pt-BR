# Comparar feeds de dados no Customer Journey Analytics e no Adobe Analytics

Os feeds de dados no Customer Journey Analytics e no Adobe Analytics permitem exportar dados brutos para plataformas de terceiros. Se você usou feeds de dados anteriormente no Adobe Analytics, use as seguintes informações para entender as diferenças nos recursos disponíveis, no comportamento de processamento de dados e muito mais.

## Funcionalidade Comparar feeds de dados

| **Opções e conceitos de configuração** | **Descrição** | **Customer Journey Analytics** | **Adobe Analytics** |
|---------|----------|---------|---------|
| Entrada de dados | O tipo de dados que pode ser coletado e incluído nos feeds de dados. | Suporta entrada de dados entre canais, incluindo dados da Web, dados da central de atendimento, dados de pontos de venda e muito mais. | Oferece suporte somente à entrada de dados da Web e móveis. |
| Processamento de dados | Os dados geralmente precisam ser processados antes de serem úteis para relatórios, e esse processamento afeta a funcionalidade dos feeds de dados. Os dados são processados em estágios diferentes, dependendo se você está usando o Customer Journey Analytics ou o Adobe Analytics. | Os dados são processados no momento do relatório e, portanto, muitos recursos de relatórios no Customer Journey Analytics podem ser usados para alterar dados históricos, como compilação, campos derivados, Preparo de dados, segmentação e a janela de relatórios. | Os dados são processados no momento da coleta e, portanto, os recursos de relatório, como regras de processamento, regras VISTA, compilação e segmentação, não afetam os dados históricos. |
| Janela de feed de dados / Frequência de entrega | Determina:<ul><li>Frequência com que o feed de dados é enviado</li><li>A janela de tempo incluída no feed de dados<p>Em outras palavras, depois que a janela do feed de dados terminar e ele for enviado, uma nova janela de feed de dados será iniciada.</p></li></ul><p>As seguintes opções estão disponíveis:</p><ul><li>**Diariamente**: os feeds contêm dados de um dia inteiro, da meia-noite à meia-noite no fuso horário do conjunto de relatórios. Provavelmente, uma grande parte dos dados que você obtém em um dia ou hora pertence a esse dia ou hora. Se tudo o que você fez foi dados em lote de uma central de atendimento, isso é o que você obteria. No CJA, os dados podem entrar e não importa quando entraram.
