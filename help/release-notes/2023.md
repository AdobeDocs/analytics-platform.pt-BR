# Notas de versão do Customer Journey Analytics de 2023

## Notas de versão de janeiro de 2023 {#jan23}

| Recurso | Descrição | [Início da implantação](/help/release-notes/releases.md) | [Disponibilidade geral](/help/release-notes/releases.md) |
| ----------- | ---------- | ----- | --- |
| **Compatibilidade de array de objetos com conjuntos de dados de perfil e pesquisa** | Os conjuntos de dados de perfil e pesquisa agora são compatíveis com arrays de objetos para uso no CJA. | 11 de janeiro de 2023 | 19 de janeiro de 2023 |
| **Pastas no Espaço de trabalho** | As pastas ajudam a organizar e categorizar seus projetos para obter melhor recuperação e acesso. Além disso, uma Pasta de **[!UICONTROL empresa]** compartilhada permite que Administradores criem e compartilhem conteúdo com todos os usuários do Espaço de trabalho. [Saiba mais](https://experienceleague.adobe.com/docs/analytics/analyze/analysis-workspace/build-workspace-project/workspace-folders/about-folders.html?lang=pt-BR) | N/D | 11 de janeiro de 2023 |
| **Página de destino padrão** | A [nova página de destino](/help/getting-started/landing.md) que foi introduzida no início de 2022 se tornará a experiência padrão para todos os usuários em **11 de janeiro de 2023**. A página de destino legada será descontinuada e todos precisarão usar a nova experiência. | N/D | 11 de janeiro de 2023 |
| **Página Gerenciador de projetos descontinuada** | Com o lançamento da nova página de destino, descontinuamos o **[!UICONTROL Gerenciador de projetos]** conforme listado em **[!UICONTROL Customer Journey Analytics]** > **[!UICONTROL Componentes]**. A nova página de destino tem todas as funcionalidades da antiga página do Gerenciador de projetos e muito mais. [Saiba mais](https://experienceleague.adobe.com/docs/analytics-platform/using/cja-overview/landing.html?lang=pt-BR#deprecate-pm-page) | N/D | 11 de janeiro de 2023 |
| **Agendar pastas de trabalho no Report Builder** | No Customer Journey Analytics, você pode criar programações para enviar pastas de trabalho em intervalos regulares. Agora os recipients podem receber as atualizações mais recentes de suas pastas de trabalho regularmente. [Saiba mais](https://experienceleague.adobe.com/docs/analytics-platform/using/cja-reportbuilder/schedule-reportbuilder.html?lang=pt-BR) | N/D | 11 de janeiro de 2023 |
| **Salvar novos projetos automaticamente** | O Analysis Workspace agora salva automaticamente os projetos recém-criados. Se, por qualquer motivo, você perder inesperadamente o acesso a um projeto recém-criado antes de salvá-lo de forma manual, uma versão de recuperação do projeto agora estará disponível. Antes, os projetos eram salvos automaticamente somente depois de terem sido salvos de forma manual. [Saiba mais](/help/analysis-workspace/build-workspace-project/save-projects.md) | N/D | 11 de janeiro de 2023 |
| **Preferências do usuário avançadas** | Agora você pode configurar preferências adicionais no nível do usuário (em [!UICONTROL Componentes] > [!UICONTROL Preferências]). Ao definir a preferência do usuário, suas seleções se estendem por projetos, tabelas e visualizações. A página Preferências agora contém as novas guias a seguir, cada uma contendo várias novas opções de configuração:<ul><li>Tabela de forma livre</li><li>Visualizações&lt;/li></ul>. Além disso, agora há mais preferências disponíveis nas guias **[!UICONTROL Geral]** e **[!UICONTROL Projeto]**.<p>Anteriormente, muitas dessas preferências eram configuráveis somente para projetos, tabelas e visualizações individuais. [Saiba mais](/help/analysis-workspace/user-preferences.md) | N/D | 11 de janeiro de 2023 |

### Avisos importantes para administradores do CJA

| Aviso | Aviso adicionado ou atualizado | Descrição |
| --- | --- | --- |
| **Melhoria no mapeamento de IP para geolocalização** | 29 de setembro de 2022 | O fornecedor de pesquisas de IP da Adobe, Digital Element, está atualizando para um novo conjunto de dados aprimorado (NetAcuity Pulse) de mapeamento de IP para geolocalização. O Adobe Analytics adiou a adoção desse novo conjunto de dados para **11 de janeiro de 2023**. O novo banco de dados será mais preciso que as versões anteriores. Alguns mapeamentos de IP para geolocalização serão alterados/melhorados quando o novo banco de dados for adotado.<p> Os dados do CJA fornecidos por meio do [!UICONTROL Conector de origem do Analytics] também aproveitarão automaticamente os novos mapeamentos. |
