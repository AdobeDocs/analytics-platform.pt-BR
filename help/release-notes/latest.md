# Notas de versão atuais do Adobe Customer Journey Analytics (janeiro de 2025)

**Última atualização**: quinta-feira, 22 de janeiro de 2025

Essas notas de versão abrangem o período de lançamento de 23 de outubro de 2024 a 30 de janeiro de 2025. As versões do Adobe Customer Journey Analytics operam em um [modelo de entrega contínua](releases.md) que permite uma abordagem escalável e em fases para a implantação de recursos. Sendo assim, essas notas de versão são atualizadas várias vezes por mês. Verifique-as regularmente.

## Recursos novos ou atualizados

| Recurso | Descrição | [Início da implantação](releases.md) | [Disponibilidade geral](releases.md) |
| ----------- | ---------- | ------- | ---- |
| **Experiência de uso de Conexões atualizada** | A guia **[!UICONTROL Uso]** em Conexão agora fornece visualizações aprimoradas para estes tipos de linhas reportáveis: dados principais, assimilados e históricos. Você também pode visualizar e analisar os dados de uso por conexão, conjunto de dados, sandbox ou tag. [Saiba mais](https://experienceleague.adobe.com/en/docs/analytics-platform/using/cja-connections/manage-connections#connections-usage) |  | quinta-feira, 15 de janeiro de 2025 |
| **API para migrar projetos Adobe Analytics e quaisquer componentes incluídos para Customer Journey Analytics** | Uma API agora está disponível para migrar os projetos do Adobe Analytics e os componentes incluídos para o Customer Journey Analytics. Anteriormente, a migração de projetos e componentes estava disponível somente por meio da interface do usuário. [Saiba mais](https://adobedocs.github.io/analytics-2.0-apis/?urls.primaryName=CJA%20Migration%20APIs). Selecione **APIs de migração do CJA** na lista suspensa. |  | quinta-feira, 15 de janeiro de 2025 |
| **Usar modelos personalizados do Customer Journey Analytics na página Relatórios do Journey Optimizer** | Agora é possível personalizar a nova interface de relatórios no Adobe Journey Optimizer criando ou editando um modelo no Customer Journey Analytics e salvando o modelo a ser usado na página Relatórios no Journey Optimizer. Anteriormente, a nova interface de relatórios no Adobe Journey Optimizer não podia ser personalizada. <p>Para obter mais informações, consulte &quot;Criar um modelo&quot; ou &quot;Editar ou excluir um modelo&quot; em [Criar e gerenciar modelos](https://experienceleague.adobe.com/en/docs/analytics-platform/using/cja-workspace/templates/create-templates).  |  | quinta-feira, 15 de janeiro de 2025 |
| **Modelos no Analysis Workspace** | Os modelos agora estão disponíveis no Customer Journey Analytics.<ul><li>**Modelos pré-criados**: uma grande seleção de modelos pré-criados está disponível. Você pode usar esses modelos para obter insights rápidos sobre os cenários de relatórios mais comuns. Modelos pré-criados podem ser usados como estão. Ou podem ser usadas como ponto de partida para um projeto, que pode ser personalizado para se adequar melhor a uma finalidade específica. [Saiba mais](/help/analysis-workspace/templates/use-templates.md)</li><li>**Modelos de empresa**: os administradores podem criar modelos de empresa para atender às necessidades de casos de uso específicos de sua organização. Os modelos de empresa criados pelos administradores estão disponíveis para os usuários em sua organização. [Saiba mais](/help/analysis-workspace/templates/create-templates.md)</li></ul> | Janeiro de 15 | sexta-feira, 30 de janeiro de 2025 |
| **Uso do produto** | Veja como sua organização usa o Customer Journey Analytics. Habilitar esse recurso cria um conjunto de dados na Adobe Experience Platform que coleta dados quando qualquer pessoa na organização usa o Analysis Workspace. Uma conexão e uma exibição de dados também são criadas automaticamente, fornecendo acesso a dimensões como os principais tipos de projeto, usuários mais ativos e componentes mais populares usados em projetos. [Saiba mais](/help/tools/product-usage/usage-overview.md) | 23 de outubro de 2024 | 22 de janeiro de 2025 |
| **Legendas inteligentes v2** | As legendas inteligentes agora são compatíveis com as seguintes visualizações: Multilinha, Barra, Barra horizontal, Rosca, Área, Fluxo e Fallout. É possível optar por mostrar todas as legendas inteligentes de uma só vez em uma exibição expandida ou mostrar legendas inteligentes individuais em uma exibição individual. [Saiba mais](https://experienceleague.adobe.com/en/docs/analytics-platform/using/cja-workspace/visualizations/intelligent-captions) |  | 22 de janeiro de 2025 |
| **Adicionar análises guiadas aos projetos a partir da Análise Guiada** | Permite adicionar análises guiadas a projetos Workspace a partir da Análise guiada. Também é possível adicionar análises guiadas diretamente no Analysis Workspace. [Saiba mais](https://experienceleague.adobe.com/pt-br/docs/analytics-platform/using/guided-analysis/overview) |  | 22 de janeiro de 2025 |
| **Coleção de mídia: atualizações do Adobe Source Connector para o novo Media Reporting XDM** | O Analytics Source Connector mapeará automaticamente os dados de streaming de mídia no Adobe Analytics para os mesmos campos usados pelo Web SDK. Atualmente, os dados são mapeados para os locais antigo e novo, mas somente o novo local será usado no futuro. (Link da documentação em breve) |  | sexta-feira, 30 de janeiro de 2025 |
