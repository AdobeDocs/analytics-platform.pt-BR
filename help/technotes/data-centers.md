# Locais de hospedagem do Customer Journey Analytics

O Adobe Customer Journey Analytics é hospedado em data centers corporativos de provedores de serviços de nuvem pública na América do Norte, Europa e APAC.

Após o provisionamento, os clientes designam a região em que seus dados do Adobe Experience Platform residirão. Os dados assimilados na Customer Journey Analytics a partir do data lake da Adobe Experience Platform serão armazenados na mesma região.

Consulte [Coleta de dados regionais](https://experienceleague.adobe.com/pt-br/docs/core-services/interface/data-collection/rdc) na documentação da Adobe Experience Cloud para obter mais informações.


## Exibir o data center onde seus dados estão armazenados

>[!NOTE]
>
>Não é possível mover dados de um data center para outro.

Para exibir em qual data center seus dados estão armazenados:

1. Faça logon na [Adobe Experience Cloud](https://experience.adobe.com).

1. Selecione **[!UICONTROL Experience Platform]** no alternador de aplicativos ![App](/help/assets/icons/Apps.svg) na parte superior direita da interface.

1. O código de região do data center atribuído é mostrado na parte superior direita do Experience Platform.

   ![Data center na home page do Experience Platform](assets/experience-platform-data-center.png)

1. Use a tabela a seguir para entender a área geográfica à qual seu código de região está associado:

   | Código da região do Adobe | Provedor de nuvem | Região geográfica |
   |-------------------|-------|-------------------------------------------|
   | VA7 | Azure | EUA (padrão) |
   | VA6 | AWS | EUA (por solicitação) |
   | NLD2 | Azure | Holanda, Amsterdã |
   | CAN2 | Azure | Canadá Central, Toronto |
   | AUS5 | Azure | Austrália, Sydney |
   | GBR9 | Azure | Reino Unido, Londres |
   | IND2 | Azure | Índia |
   | CHE2 | Azure | Suíça |
