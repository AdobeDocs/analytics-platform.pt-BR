# [!UICONTROL Painéis] do Adobe Analytics - Visão geral

Os **painéis do [!UICONTROL Adobe Analytics]** fornecem insights a qualquer hora e em qualquer lugar do Adobe Analytics. O aplicativo permite aos usuários acesso móvel a scorecards intuitivos. Os scorecards são uma coleção das métricas principais e de outros componentes apresentados em um layout lado a lado, que você pode usar para obter detalhamentos mais minuciosos e relatórios de tendências. O aplicativo móvel é compatível com os sistemas operacionais iOS e Android.

Para saber mais sobre os [!UICONTROL painéis] do Adobe Analytics, consulte os seguintes recursos:

* [Recursos](/help/mobile-app/curator.md) do curador: Destinado a ajudar os curadores de dados a configurar scorecards para os usuários executivos nos  [!UICONTROL painéis]. Os curadores podem ser administradores organizacionais ou pessoas em outras funções responsáveis pela configuração dos Scorecards:

   * [Criar scorecards](/help/mobile-app/create-scorecard.md)
   * [Auxiliar executivos com a configuração do aplicativo e o uso do aplicativo](/help/mobile-app/set-up-execs.md)
   * Para assistir a uma visão geral em vídeo da experiência de preparação, consulte [Construtor de scorecard dos painéis do Adobe Analytics](https://experienceleague.adobe.com/docs/analytics-learn/tutorials/additional-tools/analytics-dashboards/adobe-analytics-dashboards-scorecard-builder.html?lang=pt-BR).


* [Guia de início rápido do usuário executivo](/help/mobile-app/executive.md): este guia tem como objetivo ajudar usuários executivos a ler e interpretar Scorecards nos [!UICONTROL painéis]. O aplicativo permite que os usuários executivos visualizem uma ampla renderização de importantes dados resumidos de maneira rápida e fácil em seus próprios dispositivos móveis.

   * Para assistir a um vídeo que destaca essas informações, consulte [Experiência no aplicativo de painéis do Adobe Analytics](https://experienceleague.adobe.com/docs/analytics-learn/tutorials/additional-tools/analytics-dashboards/adobe-analytics-dashboards-in-app-experience.html?lang=pt-BR).
